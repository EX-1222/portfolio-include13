## Sobre este repositorio

Este repositorio recoge mi portfolio técnico de ciberseguridad y documentación profesional desarrollada durante la formación.

El objetivo es organizar evidencias, prácticas, proyectos y aprendizajes relacionados con redes, sistemas, soporte IT, análisis de seguridad, automatización y respuesta ante incidentes.


## Proyectos

| Proyecto | Tecnologías | Descripción |
|---|---|---|
| MIDTECH | n8n, Google Drive, OpenAI, RAG | Asistente de ciberseguridad y compliance para consultar políticas y normativas |
| Análisis de políticas | OpenAI, Markdown, Google Drive | Revisión de la política de seguridad de MIDTECH y detección de áreas cubiertas o ausentes |
| Gap Analysis normativo | n8n, OpenAI, GDPR, ISO 27001, ENS | Comparación de la política de MIDTECH con requisitos normativos |
| Trazabilidad de evidencias | Markdown, GitHub, capturas | Registro de pregunta, documento consultado, fragmento recuperado, respuesta generada y captura asociada |

## Habilidades técnicas


- Control de versiones: Git y GitHub.
- Documentación técnica en Markdown.
- Uso de OpenAI/GPT-4o aplicado a análisis documental.
- Automatización básica con n8n.
- Organización documental en Google Drive.
- Conceptos de RAG para recuperar fragmentos relevantes.
- Ciberseguridad y compliance: GDPR, ISO 27001 y ENS.
- Análisis de políticas de seguridad.
- Trazabilidad de evidencias técnicas.
