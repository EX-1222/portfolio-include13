# Proyecto MIDTECH

## Descripción

Asistente de ciberseguridad y compliance basado en IA para apoyar la consulta de documentación interna y normativa.

El sistema está diseñado para analizar la política de seguridad de MIDTECH, comparar requisitos con GDPR, ISO 27001 y ENS, responder preguntas de auditoría y proponer mejoras priorizadas.

## Tecnologías utilizadas

- OpenAI / GPT-4o
- n8n
- Google Drive
- RAG
- Markdown
- GitHub

## Capacidades del asistente

1. Análisis de políticas.
2. Detección de riesgos e incumplimientos.
3. Preguntas de auditoría.
4. Propuestas de mejora.

## Trazabilidad

Cada prueba deberá relacionar:

Pregunta realizada → documento consultado → fragmento recuperado → respuesta generada → captura asociada.
